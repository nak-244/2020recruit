募集要項

採用学部・学科 全学部・全学科
募集職種
企画営業
大手通信・物流・イベント・外食産業など業界業種を問わず、幅広くアウトソーシングプランを提案するコンサルティング志向の営業です。クライアントの経営・人事戦略を掌握し、人材面を中心としたアウトソーシングプランを提案することで経営の効率化を提供します。常にクライアント・経営者の立場でニーズを探求する為、経営感覚も自然に身に付きます。
コーディネーター
人材ビジネスで「人」はまさしく「人財」です。登録スタッフ様も当社の大切なお客様です。スタッフ様の適性・経験を考え人材配置をコントロールし、企業とスタッフ様のベストマッチングを担う重要な業務です。
スーパーバイザー
テレマーケティング・コールセンターや、派遣先企業内でのスーパーバイジング業務をお願いします。クライアント⇔スタッフ様⇔当社をWIN＝WINの関係として信頼関係を構築、取引の拡大を目指します。クライアントへの提案・相談・折衝、業務フローの設計・進行管理、スタッフ様の教育研修・マニュアルの作成等、マネジメント全般を最終的にお願いします。今需要が非常に高い、スーパーバイジング業務のスキルが身に付きます。
